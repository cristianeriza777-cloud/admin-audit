== Description ==

Lightweight administrator visibility and suspicious admin monitoring for WordPress.

PlusExpert Admin Audit improves visibility over sensitive administrator activity in WordPress.

The plugin helps website owners detect suspicious administrator accounts, review sensitive access changes, and monitor important admin-related events.

Designed for lightweight monitoring without turning WordPress into a heavy security suite.

== Main Features ==

* Detect newly created administrator accounts
* Identify potentially suspicious administrator patterns
* Review sensitive admin-related activity
* Lightweight and easy to use
* Simple and clean dashboard
* Designed using real-world hacked site scenarios

== Free Version ==

PlusExpert Admin Audit is a fully functional free plugin. It includes core admin visibility, auditing tools, suspicious admin detection, hidden admin detection, admin repair tools, and a complete action history log.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plusexpert-admin-audit/` directory, or install the plugin through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Open the Admin Audit section from the WordPress admin menu.

== Frequently Asked Questions ==

= Does this plugin remove malware? =

No. This plugin focuses on administrator visibility, monitoring, and alerts.

= Does this plugin include a firewall? =

No. The goal is lightweight admin auditing and suspicious administrator detection.

= Does the free version work on its own? =

Yes. The plugin is fully functional and provides admin scanning, suspicious admin detection, hidden admin detection, and admin repair tools.
